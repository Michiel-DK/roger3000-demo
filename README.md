# roger3000 — investor-intelligence demo

A scrollytelling product demo for **roger3000**, an AI engine that runs investor
intelligence on a client's own corpus — and audits every claim it writes against
a source before you see it.

**▶ Live demo:** https://michiel-dk.github.io/roger3000-demo/

Told in five moments — ingest → intelligence → trust → outreach → flywheel — each
a self-animating product mock. Built as a single self-contained page: inline
CSS + vanilla JS, zero dependencies, no build step, scroll-triggered animation.

_All names, firms, people and figures on the page are generated demo data
(fictional client "Meridian Advisory"). The only real numbers are the three
claim-auditor calibration figures._
